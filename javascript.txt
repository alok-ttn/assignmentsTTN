Js uses a feature called lexical scoping, means variable defined outside function are automatically available inside.
Now if you want to look inside the function object cause fuctions are also objects in js.
Use dir to look into objects.




var passed=4;
var addTo =function()
{
var inner=2;
return passed +inner;
};
console.dir(addTo);
